# How to Start Your Server

## ⚠️ IMPORTANT: Always run from the project directory!

Your server must be started from the project folder.

## Quick Start Commands:

```bash
cd "/Users/alexispinzon/Library/Mobile Documents/com~apple~CloudDocs/CURSOR PROYECECT ONE"
python3 server.py
```

## What You'll See:

When the server starts successfully, you'll see:
```
 * Running on http://127.0.0.1:5000
```

Then open your browser to: `http://localhost:5000`

---

## Full Command (Copy/Paste This):

```bash
cd "/Users/alexispinzon/Library/Mobile Documents/com~apple~CloudDocs/CURSOR PROYECECT ONE" && python3 server.py
```

---

## Stop the Server:

Press `Ctrl + C` in the terminal where the server is running.
